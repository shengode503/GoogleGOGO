# Note
  - Input      : t, s
  - Output     : True if t is s's anagram else False

# Brute-force
  - For loop to iter s and another for loop to iter t and compare the character, if the same, remove the character from t. 
  - Complexity: O(n^2), O(1)

# How to better resolve it? 
  - Create a list of size 26 to count the frequency of the charactor.
  - Iterate both s and t, for s character, + 1 for the index of the character, for t character, -1 for the index of the character.
  - sum the list, if the summary is 0, return True, else False.
  - Complexity: O(n), O(1)

# Coding
def solve(t, s):
    if len(s) != len(t):
        return False

    count_list = [0] * 26
    for i in range(len(t)):
        s_char, t_char = ord(s[i]) - ord('a'), ord(t[i]) - ord('a')

        count_list[s_char] += 1
        count_list[t_char] -= 1
    return sum(count_list) == 0

# Dry run
# t = 'abc', s = 'cab'
#   - i=0, s_char, t_char = 2, 0, count[2] = 1, count[0] = -1
#   - i=1, s_char, t_char = 0, 1, count[0] = -1 + 1 = 0, count[1] = -1
#   - i=2, s_char, t_char = 1, 2, count[1] = -1 + 1 = 0, count[2] = 1 - 1 = 0

#

#   - i=0, s_char, t_char = 0, 0, count[0] = 1 - 1 = 0
